# 智巡草原（Ranch Edge Inspection Bot）

智巡草原是一个面向牧区物联网终端的移动端无人值守巡检系统。项目基于边缘计算架构，利用异构设备控制与视觉感知能力，对现有牧场管理 App 进行非侵入式自动巡检、状态提取和异常告警。

## 项目定位

1. 适用对象：中小牧场、合作社、试验牧场。
2. 核心价值：不改造原硬件、不依赖私有 API、低成本快速上线。
3. 技术路径：边缘节点本地识别与判断，弱网环境下持续运行。

## 核心能力

1. 异构设备统一纳管：支持对多品牌安卓终端进行连接、控制与会话管理。
2. 非侵入式界面语义理解：通过 OCR 提取页面关键指标并进行规则判断。
3. 弱网自愈：支持心跳保活、断线重连、异常回退等稳定性机制。
4. 分布式集群管理：单主机可并发管理多个巡检实例，支持按点位隔离配置。
5. 多通道告警：支持企业微信与钉钉机器人接入。

## 系统架构

1. 终端层：安卓手机/平板，运行业务 App。
2. 边缘层：工控机/树莓派，运行本项目 Python 引擎。
3. 协同层：企业微信/钉钉，用于异常通知与运维闭环。

## 代码模块映射

1. 设备控制：adb_controller.py
2. 设备发现：emulator_discovery.py
3. 主流程编排：main_adb.py
4. 视觉识别：simple_ocr.py
5. 控制台与多实例入口：gui_launcher.py
6. 截图/输入通道扩展：nemu_ipc.py、adb_tools/、platform-tools/
7. 调试与诊断：woa_debug.py

## 运行环境

1. 操作系统：Windows 10 / 11
2. Python：建议使用项目虚拟环境 .venv
3. 依赖安装：

```powershell
pip install -r requirements.txt
```

## 快速开始

1. 启动安卓终端并打开目标业务 App。
2. 激活虚拟环境后启动图形界面：

```powershell
python gui_launcher.py
```

3. 在界面中扫描设备并连接目标终端。
4. 完成 OCR 与告警参数配置后启动巡检流程。
5. 如需并发巡检，可启用多实例并绑定独立配置文件。
